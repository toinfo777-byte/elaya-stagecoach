# Elaya Agent · Roadmap (v0.1 — v1.0)
Документ: docs/technical/Elaya_Agent_Roadmap.md  
Статус: v0.1 (living document)  
Цикл: Q4 2025 — Q2 2026

---

## 1. Миссия Roadmap
Roadmap определяет стратегию развития Elaya Agent —
локального операционного слоя Элайи, который объединяет:

- локальную файловую систему  
- git  
- web-core (Render)  
- операционные процессы HQ  
- тренер-ботов  

Это путь к автономной, симбиотической системе разработки.

---

# 2. Этапы развития (мапа версий)

## ⭐ v0.1 — Прототип (завершено)
- cli.py как единая точка входа  
- команды: status / project / diag  
- минимальный показ root, папок и python  
- базовая диагностика venv  

## ⭐ v0.2 — Операционная база (текущая)
- рефакторинг структуры  
- единый стиль вывода  
- диагностика окружения  
- стандартизация команд  
- документация v0.2  
- подготовка к layers (commands/*.py)

---

## ⭐ v0.3 — File & Repo Inspector
Фокус: анализ структуры проекта.

Функциональность:
- elaya structure  
- определение “мёртвых” файлов  
- анализ импортов  
- навигация (jump to, list modules)  
- детектор дублирующих файлов  
- быстрые шаблоны Markdown  
- utils/fs.py и utils/repo.py  

Интеграция:
- отображение дерева проекта  
- метрики структуры  

---

## ⭐ v0.4 — Web-Core Sync & Pulse
Фокус: подключение к ядру Элайи.

Функции:
- elaya sync  
- получение /api/status  
- отправка /api/event  
- локальный Pulse  
- health-монитор web-core  
- состояние Render-сервиса  

---

## ⭐ v0.5 — Operator Mode (защищённый)
Фокус: первые мутации проекта.

Функции:
- git add / commit / push  
- автоматизация коммитов  
- elaya fix imports  
- elaya branch create/switch  
- защита GUARD_KEY  
- операторское управление проектом  

---

## ⭐ v0.6 — Documentation Engine
Фокус: создание и поддержа документации.

Функции:
- генерация .md файлов  
- обновление TOC  
- автоматические технические отчёты  
- интеграция с HQ Pulse  

---

## ⭐ v0.7 — AI-Augmented Developer
Фокус: симбиоз ИИ + человек.

Функции:
- подсказки по кодовой базе  
- контекстный анализ изменений  
- предсказание ошибок  
- локальная LLM-поддержка (в будущем)  

---

## ⭐ v1.0 — Полноценный Операционный Слой
Фокус: стабильность, автономность, симбиоз.

Функции:
- полный Operator Mode  
- автоматическое восстановление связей  
- адаптивная диагностика  
- системная интеграция с HQ и тренером  
- самодокументирование  

---

# 3. Принцип Roadmap
Roadmap не обязует — он направляет.

Он задаёт:

- ритм  
- поступательность  
- ясность будущего  
- вектор архитектуры  

Элайя растёт через различение и соприсутствие.  
Agent — её орган на локальной машине.
